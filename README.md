## Install

[npm][]:

```sh
npm install lipo-express
```

[yarn][]:

```sh
yarn add lipo-express
```


## Usage

```js
const LipoExpress = require('lipo-express');

const lipoExpress = new LipoExpress();

console.log(lipoExpress.renderName());
// script
```
